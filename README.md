## Introducción

Este repositorio no solo funciona como un registro académico, sino también como una **guía de consulta personal y divulgativa**, que puede servir a otros estudiantes interesados en comprender los principios y metodologías del desarrollo de software.

Cada apartado contiene explicaciones técnicas, apuntes teóricos, diagramas, ejercicios prácticos y reflexiones sobre los temas estudiados.

El contenido está estructurado para facilitar su navegación por trimestres, permitiendo seguir el avance del proceso formativo de forma cronológica y progresiva.

---

## Contexto del programa

El programa **Tecnólogo en Análisis y Desarrollo de Software** del SENA está enfocado en la creación de aplicaciones informáticas de acuerdo con requerimientos funcionales y técnicos, promoviendo el trabajo en equipo, la ética profesional y la innovación.

### Objetivo

Desarrollar habilidades para el análisis, diseño, construcción, implementación y control de calidad de soluciones de software, aplicando metodologías, estándares y tecnologías actuales, con un enfoque en la resolución de problemas del sector productivo y social.

### Perfil de egreso

- Técnico en Tecnologías de la Información
- Codificador de aplicaciones informáticas
- Competente en el uso de herramientas de desarrollo, bases de datos, control de versiones y metodologías ágiles

### Competencias técnicas centrales

1. Establecer requisitos de la solución de software de acuerdo con estándares y procedimiento técnico.
2. Evaluar requisitos de la solución de software de acuerdo con metodologías de análisis y estándares.
3. Estructurar propuesta técnica de servicio de tecnología de la información según requisitos técnicos y normativa.
4. Diseñar la solución de software de acuerdo con procedimientos y requisitos técnicos.
5. Desarrollar la solución de software de acuerdo con el diseño y metodologías de desarrollo.
6. Implementar la solución de software de acuerdo con los requisitos de operación y modelos de referencia.
7. Controlar la calidad del servicio de software de acuerdo con los estándares técnicos.

Estas competencias se abordan de manera progresiva a lo largo de los trimestres, complementándose con competencias transversales como inglés, ética, y cultura física.

## Metodología y herramientas

A lo largo del programa se han utilizado diversas herramientas y metodologías para facilitar el aprendizaje y la gestión de proyectos:

- **Documentación y ofimática:** Google Drive (Docs, Sheets, Slides)
- **Gestión de proyectos:** JIRA (seguimiento de tareas y sprints)
- **Control de versiones:** Git y GitHub (repositorios públicos y privados)
- **Entornos de desarrollo:** Visual Studio Code, XAMPP, MySQL Workbench, SQL Server
- **Lenguajes y tecnologías:** Python, JavaScript, HTML5, CSS3, SQL, entre otros

Este repositorio refleja el uso de estas herramientas, integrando el conocimiento teórico con la práctica constante.
