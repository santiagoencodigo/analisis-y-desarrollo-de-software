# Social Cards

> Práctica de tarjetas de perfil de usuario · WorldSkills 2025  
> Primer trimestre ADSO

## Contexto WorldSkills

El objetivo era aprender a organizar información dentro de `divs` y aplicar estilos de tarjetas similares a las de redes sociales. Fue mi primer acercamiento a la disposición de elementos (avatar, nombre, usuario, estadísticas) sin usar frameworks.

## Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox incipiente, sombras, bordes redondeados)

## Aprendizajes clave

- Uso de `div` contenedor y elementos internos.
- Selectores CSS anidados para estilizar componentes.
- Centrado de elementos con `margin: auto` y `text-align`.
- Comprensión de la herencia de estilos.
