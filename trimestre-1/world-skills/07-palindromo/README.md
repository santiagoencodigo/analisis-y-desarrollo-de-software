# Palíndromo

> Validador de palabras o frases palíndromas · WorldSkills 2025 (actividad de competencia)

## Contexto WorldSkills

Este ejercicio **apareció en la competencia** y fue uno de los primeros que requirió **lógica de programación** (JavaScript). Aprendí a manipular strings, usar condicionales y actualizar el DOM. Fue mi primer "algoritmo" real.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (manipulación de cadenas, eventos)

## Aprendizajes clave

- Limpiar cadenas (eliminar espacios, signos, mayúsculas).
- Usar `split()`, `reverse()`, `join()` para invertir una cadena.
- Comparar la cadena original con la invertida.
- Escuchar eventos `click` y mostrar resultados dinámicamente.
