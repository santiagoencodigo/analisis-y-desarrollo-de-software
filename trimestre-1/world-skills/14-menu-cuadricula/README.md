# Menú Cuadrícula – Conciertos Colombia

> Práctica de Grid y Flexbox para organizar secciones · WorldSkills 2025

## Contexto WorldSkills

Este proyecto fue importante para **aprender a ubicar piezas** en una cuadrícula (menú de navegación, tarjetas de artistas, locaciones, etc.). Aunque personalmente no me gustó el resultado visual, fue un excelente ejercicio para dominar CSS Grid y Flexbox.

## Tecnologías utilizadas

- HTML5
- CSS3 (Grid, Flexbox)

## Aprendizajes clave

- Crear un layout con `display: grid` y `grid-template-areas`.
- Alinear elementos dentro de cada celda con Flexbox.
- Combinar ambas técnicas para un diseño coherente.
- Entender que la funcionalidad es más importante que la belleza en etapas de aprendizaje.
