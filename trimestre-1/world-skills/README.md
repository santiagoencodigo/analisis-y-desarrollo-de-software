## Introducción

Esta sección documenta de manera viva mi proceso de formación durante el primer trimestre del programa ADSO (Análisis y Desarrollo de Software), en paralelo a mi participación en las actividades de **WorldSkills Colombia**.  
La página principal funciona como un archivo académico personal que reúne nueve proyectos, cada uno reflejando un concepto o técnica aprendida por primera vez.

---

## Contexto WorldSkills

En junio de 2025, mientras cursaba mi primer trimestre, tuve la oportunidad de compartir espacios de codificación con compañeros de cuarto trimestre. WorldSkills se convirtió en un escenario de retos colaborativos.  
Allí escribí mis primeras líneas de HTML, descubrí el poder de CSS para diseñar layouts, y empecé a entender cómo JavaScript da vida a una interfaz.

Esta sección es un **homenaje a ese punto de partida**: la curiosidad, el vértigo inicial y la alegría cuando un botón finalmente respondía.

---

## Objetivos del portafolio

- **Registrar** mis primeros pasos en programación web.
- **Mostrar** la evolución desde ejercicios simples (palíndromos) hasta interacciones complejas (Drag & Drop, Simon Say).
- **Servir** como memoria visual y técnica de los fundamentos: lógica, eventos, DOM, animaciones.
- **Inspirar** a otros estudiantes que empiezan su camino en ADSO.

---

## Tecnologías utilizadas

- **HTML5** – Estructura semántica, accesibilidad.
- **CSS3** – Flexbox, Grid, variables, keyframes, media queries.
- **JavaScript (Vanilla)** – Manipulación del DOM, eventos, objetos Date, bucles, condicionales, Drag & Drop nativo.

No se utilizan frameworks ni librerías externas para preservar la esencia del aprendizaje puro.

## Proyectos desarrollados

### 1. Palíndromo
Validador de palabras o frases palíndromas.  
🔹 *Aprendizajes:* manipulación de strings, condicionales, eventos.

### 2. Fecha y Hora
Reloj y calendario en tiempo real con actualización dinámica.  
🔹 *Aprendizajes:* objeto `Date`, funciones y refresco del DOM.

### 3. WS CSS
Laboratorio de estilos: Flexbox, Grid, animaciones CSS, diseño responsivo.  
🔹 *Aprendizajes:* layout moderno, `@keyframes`, transiciones.

### 4. Simon Say
Juego de memoria interactivo (versión inspirada en tutorial).  
🔹 *Aprendizajes:* integración completa HTML/CSS/JS, estados de juego.

### 5. Almacén de Cajas
Interfaz Drag & Drop para mover elementos entre zonas.  
🔹 *Aprendizajes:* eventos del mouse, manipulación dinámica de elementos.

### 6. Rompecabezas Módulo A
Puzzle con imágenes alusivas a WorldSkills 2024 usando arrastrar y soltar.  
🔹 *Aprendizajes:* organización lógica + reordenación visual.

### 7. Tablas de Multiplicar
Generador de tablas a partir de un número ingresado.  
🔹 *Aprendizajes:* bucles `for`, renderizado de HTML desde JavaScript.

### 8. Órbita Celestial
Planetario animado solo con HTML y CSS (transformaciones, rotaciones).  
🔹 *Aprendizajes:* posicionamiento absoluto, creatividad visual con CSS puro.

## Aprendizajes obtenidos

Este primer trimestre fue una inmersión en el pensamiento lógico y la construcción de interfaces. Algunas reflexiones:

- **HTML:** aprendí que el marcado no es "diseño" sino arquitectura de contenido.
- **CSS:** pasé del miedo a las propiedades a disfrutar Grid y Flexbox.
- **JavaScript:** entendí que los eventos y el DOM son el puente entre la intención y la acción.
- **Depuración:** cada error me enseñó a leer la consola y a ser persistente.
- **Drag & Drop:** comprender el flujo de eventos nativos amplió mi visión sobre la interactividad.
- **Animaciones:** las transiciones suaves hacen que una interfaz se sienta viva.
